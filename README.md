Проєктування вимірювальної апаратури часто вимагає знати яка є або як змінюється температура на окремих елементах приладу.
Дистанційний вимірювач температури дозволяє вимірювати температуру на окремих елементах або ділянках  приладу.
Вимірювач створено на МК STM32F103, TFT LCD 1.44" 128x128, MLX90614
